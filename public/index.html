Control de rotaciones entre amigos

Esta aplicación web sirve para controlar las rotaciones entre amigos como, por ejemplo, el turno de pago para los desayunos entre compañeros de trabajo. Debo reconocerlo, esta es la principal motivación para realizar el desarrollo de esta aplicación, preservar la amistad de los compañeros de trabajo.

Desayunos

Tenéis problemas a la hora de saber a quién le toca el turno para pagar el desayuno. Aquí tienes la solución. Un compañero/amigo gestiona el grupo de desayunos y los demás lo fiscalizan.

Guardias

No os pasa que perdéis la capacidad de recordar quien fue la última o antepenúltima persona a la que le toco la guardia. Pues ya sabéis, aquí está la solución.

Medio de transporte

Quien fue el último que tuvo que conducir mientras los demás amigos estaban un poquito indispuestos por los efectos del alcohol. Aquí está la solución.

Sin límite de uso

El título en realidad no es el adecuado, porque yo no le pongo ningún problema al uso de la aplicación, pero esta aplicación web está alojada en los servidore gratuitos de Heroku. Esto implica que tengo unas horas de uso al mes, 550 dynos, cuando esas horas de uso se terminan el servicio se corta hasta el siguiente mes. También existe un límite de almacenamiento en bases de datos, 50 MB, una vez alcanzado ese máximo no podré seguir dando el servicio. Así que esas son las restricciones de uso que vienen impuestas desde el exterior. Solo me queda recordaros que por mi parte no hay publicidad, no comercializo con los datos ni cobro por el servicio.

Tecnologías utilizadas

En este desarrollo utilizamos varias tecnologías Open Source. Como lenguaje de desarrollo principal utilizamos JavaScript, tanto en la parte frontal como en el servidor. En la parte del Backend utilizamos NodeJS para el entorno de ejecución de JavaScript, npm para la gestión de paquetes, Handlebars para la construcción dinámica de páginas HTML y Postgresql como servidor de bases de datos. En la parte del Frontend, utilizamos HTML y CSS como lenguaje de maquetado, Bootstrap como framework de CSS, JavaScript para la manipulación de la página web, la libreria Socket.io para la comunicación con el servidor y diversas tecnologías más. Con ello conseguimos una experiencia de usuario dinámica y amigable.
